# homebrew-tap

Official Homebrew tap for [PickFu CLI](https://github.com/pickfu/pickfu-cli).

## Install

```bash
brew install pickfu/tap/pickfu
```

## Upgrade

```bash
brew upgrade pickfu
```

## Other install methods

- npm: `npm install -g @pickfu/cli`
- curl: `curl -fsSL https://agents.pickfu.com/install.sh | bash`
- npx (no install): `npx @pickfu/cli`
